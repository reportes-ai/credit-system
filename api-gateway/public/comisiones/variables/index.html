Mantenedor Variables Comisiones

Edita las tasas, umbrales y pesos del modelo de cálculo de incentivos. Los cambios aplican al próximo cálculo.

Lógica del modelo
1
Incentivo Base = Monto≤24 × % base ≤24 + Monto>24 × % base >24 — solo si total mes ≥ Mínimo monto
2
Cruces de seguros: Cesantía ≥ Umbral cesantía y Reparaciones ≥ Umbral rep. — si no cumple, su aporte es 0
3
Factor ajuste = (Cruce cesantía × Peso ces. + Cruce rep. × Peso rep. + Calidad × Peso calidad) × Factor máx.
4
Incentivo Final = Base + Bono Cesantía + Bono Reparaciones + Bono Calidad — cada bono usa su ajuste individual
5
Con Semana Corrida = Incentivo Final × Multiplicador S.C. (aprox.)
