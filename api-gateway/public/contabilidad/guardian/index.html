Las reglas de combinación saltan cuando un mismo comprobante usa una cuenta que empieza con algún prefijo de "Cuentas A" y otra con alguno de "Cuentas B" (prefijos separados por coma). Las reglas de sistema tienen lógica propia: solo se activan/desactivan o se les cambia severidad y mensaje.
Qué afecta este mantenedor
  • Reglas de combinación y de sistema → se evalúan al grabar cada comprobante manual en Comprobantes (los asientos automáticos del motor de centralización NO pasan por el guardián: sus reglas ya están validadas en Reglas de Centralización).
  • Severidad BLOQUEA → el comprobante no se puede grabar hasta corregirlo.
  • Severidad ADVIERTE → el usuario ve la advertencia y decide contabilizar o corregir.
  • Desactivar una regla → deja de revisarse de inmediato en la digitación.
