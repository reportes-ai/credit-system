De dónde salen estos datos
Se importan desde AVSOFT: Compras → Importador/Exportador → "Exportar Documentos de Compras (1 Línea)" y Honorarios → "Movimiento Honorarios". La importación (botón arriba, también disponible en Presentación Directorio) reemplaza completos los meses que trae el archivo, así que re-importar es seguro. El export de compras no trae marca de pago: esto es el detalle de compras del período, no el saldo por pagar (ese está en el Balance, cuentas 21xx).
