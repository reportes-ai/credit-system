Qué afecta este mantenedor
  • Cuentas imputables → son las que se pueden usar al digitar Comprobantes.
  • Nombre y tipo de cuenta → cómo se muestra y clasifica cada línea en los Libros Diario y Mayor y en qué columna cae en el Balance de Comprobación (tipo INGRESO/GASTO → columnas de resultado; el resto → inventario).
  • Desactivar una cuenta → deja de aceptar movimientos nuevos, pero su historial sigue visible en libros y balance.
