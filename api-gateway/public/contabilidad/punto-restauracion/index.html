Qué afecta este mantenedor
  • Restaurar → elimina definitivamente los comprobantes con ID posterior a la marca (y sus movimientos, adjuntos y log del motor): desaparecen de Comprobantes, Libros, Balance, Estados Financieros y del Informe de Cierre.
  • NO toca Plan de Cuentas, Reglas de Centralización, Guardián, plantillas ni candados de mes.
  • Ojo: si el motor ya contabilizó eventos (pagos, ODP, remuneraciones) después de la marca, al restaurar esos asientos se pierden — el motor los re-creará solo si el evento de negocio se vuelve a gatillar (la idempotencia por origen+ref queda liberada al borrarse el comprobante). Todo queda auditado.
