Qué afecta este mantenedor
  • Cuentas y lados de cada regla → los comprobantes automáticos que aparecen en Comprobantes, Libros y Balance.
  • Interruptor Activa → si está apagado, el evento NO se contabiliza (queda registrado en el log como DESACTIVADA); el proceso de negocio sigue funcionando igual.
  • Eventos conectados hoy: pago de cuotas y prepago en Caja (por transacción), Orden de Pago marcada pagada, Castigo de saldo aprobado, cierre de mes de Provisiones, y emisión del libro de Remuneraciones.
