Qué afecta este mantenedor
• Código + glosa → el giro/actividad económica que se selecciona en Base de Proveedores y en la ficha de Clientes (datos comerciales).
• Afecto a IVA → marca si la actividad genera documentos afectos; referencia para impuestos (ver Impuestos).
• Categoría tributaria → 1ª (empresas) o 2ª (honorarios), informativa para clasificación.
• Desactivar una actividad la oculta de los selectores, sin borrar registros que ya la usaban.
