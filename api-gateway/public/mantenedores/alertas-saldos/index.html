Alertas del proceso Saldos Precio
Define quién recibe la alerta (campana) en cada paso del flujo de pago de saldos precio. Por perfil, el ejecutivo de la operación y/o personas específicas.
Qué afecta este mantenedor
  • Destinatarios por evento → generan la alerta de campana (Post Venta / Saldos a Pagar) cuando ocurre cada transición: Fondos Recibidos → Orden Emitida → Montos Cargados → Enviado a Pago → Pago Realizado.
  • Incluir ejecutivo → usa el ejecutivo que cursó la operación (vía asignación de ejecutivos del usuario).
