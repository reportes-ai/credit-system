Qué afecta este mantenedor

  • Texto de cada certificado → el cuerpo del documento en Certificados (Crédito Vigente, Prepago, Alzamiento, Pago de Cuota, Deuda Vigente, Deuda para Prepago, Preaprobado).
  • Párrafo de cierre (común) → la frase final ("Se emite el presente certificado…") que aparece en todos los certificados.
  • No cambia los datos (montos, fechas, cuotas) — esos siempre salen del crédito. Solo cambia la redacción.
