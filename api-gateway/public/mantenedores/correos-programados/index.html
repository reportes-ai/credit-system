Correos Programados
Correos automáticos por horario. Cada uno se puede activar o suspender; el envío usa la hora de Chile.
Cargando…
Qué afecta este mantenedor
• Activar/Suspender → enciende o apaga el envío automático de cada correo.
• Hora y Días → cuándo se envía (hora de Chile). Destinatarios → a quién llega (separar con comas).
• Informe Diario de Ventas → créditos otorgados acumulados del mes por Ejecutivo Comercial (mayor a menor cantidad de operaciones).
