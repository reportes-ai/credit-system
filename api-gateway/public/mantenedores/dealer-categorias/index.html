 Dealers

Programa de Segmentación de Dealers

Niveles, metas (unidades/mes) y beneficios. La meta define la categoría que el sistema propone según las ventas del mes pasado.

Qué afecta este mantenedor

  • Meta (unidades/mes) → define la Categoría Propuesta de cada dealer al pulsar "Recalcular categorías" en Dealers (cuenta créditos otorgados el mes pasado).
  • Beneficios → se muestran en el tooltip (i) junto a la Categoría Asignada en Dealers.
  • Nombre / Meta texto / Color → etiquetas y badges que se ven en la base de Dealers.
