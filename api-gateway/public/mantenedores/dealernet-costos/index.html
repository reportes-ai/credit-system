Qué afecta este mantenedor

  • Precio UF por informe → cálculo del consumo en la pestaña Informes DealerNet → Facturación (consumo = N° consultas × precio UF × valor UF del último día del mes).
  • Plan prepago contratado → base para la recomendación mantener / bajar / subir el plan según el consumo del mes cerrado.
