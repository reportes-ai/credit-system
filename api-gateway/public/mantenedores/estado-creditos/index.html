Qué afecta este mantenedor

  • Estados y transiciones (Brokerage) → dibujan el Flujo Crédito Brokerage (estados que aparecen y a qué estados puede pasar cada crédito de AutoFin/Unidad).
  • Puertas de entrada → reflejan cómo nace un crédito: Digitado (digitación o Carga Masiva) y Carta de Aprobación (vía Cartas de Aprobación).
  • Nombre y color de cada estado → etiqueta y badge con que se muestra el estado en el flujo y la documentación del proceso.
  • Ámbito → permite definir a futuro un set propio para los créditos AutoFácil (recursos propios) sin alterar el de brokerage.
  • Nota: en esta fase el mantenedor no bloquea los cambios de estado en Gestión de Créditos; es la fuente de verdad y dibuja el flujo. El bloqueo de transiciones inválidas se activará cuando el mapa esté validado.
