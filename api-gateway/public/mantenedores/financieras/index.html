Comisión Ejecutivo
Se aplica sobre el monto financiado con seguros
Qué afecta este mantenedor
  • AutoFin (spread, inscripción, limitación) → ingreso por colocación y orden de pago saldo precio en Créditos, Dashboard y Rentabilidad.
  • UAC Modelo 1 / Modelo 2 (selector "Usar este modelo") → ingreso por colocación de operaciones UNIDAD en Créditos (guardado y recálculo), snapshot de tier en Cartas de Aprobación y Dashboard. Solo un modelo rige a la vez; el Modelo 2 topa el % de las ops con plazo ≥ corte (no pagan el tier alto).
  • Al guardar cambios UAC → popup para elegir si recalcular los meses no cerrados (default: NO recalcular, aplica solo hacia adelante). Respeta campos forzados y meses cerrados.
  • % Comisión Ejecutivo → cálculo de comisión en Revisión de Comisiones.
