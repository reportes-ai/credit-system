Parámetros de Financieras

AutoFin — Fórmula de Ingreso por Tasa
AUTOFIN
Ingreso = PV(costo_fondo, plazo, PMT(tasa_cliente, plazo, monto_fin)) − monto_fin
donde: tasa_cliente = TMC_anual / 12  |  costo_fondo = TMC_mayor_200/12 − spread
Tasa máxima convencional para créditos ≤200 UF
Tasa máxima convencional para créditos >200 UF
Costo de fondo = TMC_mayor/12 − spread
UAC (Unidad de Crédito) — Fórmula de Ingreso por Colocación
UNIDAD
Ingreso = Saldo Precio × % UAC
% UAC varía según N° de operaciones otorgadas/aprobadas en el mismo mes
% sobre saldo precio — tramo bajo
% sobre saldo precio — tramo medio
% sobre saldo precio — tramo alto
Hasta este N° de ops aplica Tier 1
Hasta este N° de ops aplica Tier 2
