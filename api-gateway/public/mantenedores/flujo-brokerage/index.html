Flujo de Créditos Brokereados
AutoFácil actúa como broker en créditos AUTOFIN y UNIDAD DE CRÉDITO. No gestionamos los pagos ni el estado post-cursado — solo intermediamos y registramos el resultado final (Cursado o Desistido).
AUTOFIN
Flujo de etapas brokerage
1
Estado: CARTA_APROBACION
Carta de Aprobación
El ejecutivo genera la Carta de Aprobación y la somete a revisión. Al aprobarse se crea el crédito automáticamente en estado CARTA_APROBACION.
Carta de Aprobación impresa
2
Estado: CARTA_APROBACION → carga docs
Carga de Documentos
El Analista de Crédito carga los documentos requeridos para AUTOFIN (configurables en Tipos de Documento).
Carnet de Identidad Certificado AFP Liquidaciones + más según config.
3
Estado: OTORGADO
Envío a Autofin
Documentos completos. El expediente es enviado a Autofin para evaluación y desembolso.
✓
Estado final
Resultado Final
El Analista registra el resultado una vez confirmado por Autofin.
