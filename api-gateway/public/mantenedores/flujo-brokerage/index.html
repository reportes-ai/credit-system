Flujo por Institución
El flujo de AutoFin y Unidad de Crédito es el mismo: el de las etapas de arriba (Estados del Crédito), administrado en Estado Créditos. (Nota: lo que AutoFin llama Cursado, para nosotros es Otorgado.) El flujo operativo por institución (carga de documentos, envío a la financiera, etc.) se retomará más adelante.
