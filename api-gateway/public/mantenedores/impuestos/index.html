Impuestos
Valores paramétricos de impuestos. Todos los cálculos del sistema (IVA de comisiones, retención de boletas) leen estos porcentajes. Cambios aplican a todos los usuarios.

Qué afecta este mantenedor
  • IVA → cálculo de Neto/IVA de la participación del dealer en Cartas de Aprobación y la comisión bruta de las Cartolas. Define el monto de la Boleta (líquido = bruto / (1 + IVA)).
  • Retención de Honorarios → al autorizar una Boleta en Post Venta · Seguimiento (FACTURA RECIBIDA): retención = monto boleta × %, y el monto a depositar = boleta − retención.
