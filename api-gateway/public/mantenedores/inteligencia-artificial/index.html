Qué afecta este mantenedor

  • Interruptor maestro → habilita/inhabilita TODA la IA del sistema. Apagado, ninguna función de IA llama a Anthropic ni gasta saldo.
  • Funcionalidades con IA → prende o apaga cada análisis por separado (ej. liquidaciones, informe crediticio, carpeta tributaria). Cada una se registra sola al desarrollarse.
  • Modelo por funcionalidad → qué modelo de Anthropic usa cada análisis (más barato como Haiku vs más capaz como Opus). Viene con el recomendado; el costo se refleja en la pestaña Uso IA.
  • Mostrar logo → si el aviso y el sello incluyen el logo de Anthropic.
  • Texto mientras analiza → el aviso animado que aparece en la pantalla durante el análisis.
  • Texto del sello → la marca "Analizado con IA de Anthropic" que se estampa en el resultado o informe.

Qué afecta este mantenedor

  • Días de mes completo y Umbral de renta variable → cómo se calcula la renta en Análisis de Liquidaciones (IA).
  • Salud %, Cotización AFP base y Comisiones AFP → los chequeos de inconsistencia (salud bajo el mínimo, AFP que no cuadra) en las liquidaciones.
  • URL validador de certificados (por AFP) → el botón "Verificar folio en la AFP" que aparece al validar el certificado de cotizaciones en Evaluación Crediticia.
  • Impuesto Único 2ª Categoría (tramos) → verificación de que el impuesto retenido en cada liquidación cuadre con los tramos × UTM del mes, en Análisis de Liquidaciones (IA).
