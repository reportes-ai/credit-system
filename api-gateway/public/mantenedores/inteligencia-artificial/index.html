Qué afecta este mantenedor

  • Interruptor maestro → habilita/inhabilita TODA la IA del sistema. Apagado, ninguna función de IA llama a Anthropic ni gasta saldo.
  • Funcionalidades con IA → prende o apaga cada análisis por separado (ej. liquidaciones, informe crediticio, carpeta tributaria). Cada una se registra sola al desarrollarse.
  • Mostrar logo → si el aviso y el sello incluyen el logo de Anthropic.
  • Texto mientras analiza → el aviso animado que aparece en la pantalla durante el análisis.
  • Texto del sello → la marca "Analizado con IA de Anthropic" que se estampa en el resultado o informe.
