Potencial Dealer

Prioriza dónde poner el esfuerzo comercial según el potencial de cada dealer, y ajusta las variables del cálculo. Los datos de cada dealer (posiciones, ventas) se cargan en Dealers → Potencial Parque/Dealer.

Priorización comercial

Pareto: los dealers de más arriba concentran la mayor parte del potencial capturable → ahí van los recursos. Segmentos: el tipo de acción según cuánto vende con crédito y cuánto ya capturamos (cortes en la mediana).

Matriz de segmentos

Vertical: % que el dealer vende con crédito. Horizontal: penetración AutoFácil (cuánto de su crédito ya capturamos). Cada celda lista sus dealers ordenados por potencial.

Ranking de potencial (Pareto / cuartiles)

Ordenado por potencial absoluto = créditos que aún podrías capturar (ventas potencial − créditos actuales). Q1 = donde más conviene invertir esfuerzo comercial.

Variables del cálculo

El factor es el % de las ventas con crédito que AutoFácil podría capturar (maqueta: 50%). El diagnóstico sale por rangos del potencial.

Qué afecta este mantenedor

  • Posiciones / Ventas mensuales / Ventas con crédito → datos manuales del dealer; alimentan el cálculo de potencial. Se guardan en la ficha del dealer (Dealers).
  • Factor ventas potencial → % de las ventas con crédito que AutoFácil podría capturar (ventas potencial = ventas con crédito × factor).
  • Rangos de diagnóstico → el texto/color que se muestra según el valor del Potencial.
  • Créditos por mes → automáticos, contados de Gestión de Créditos por RUT del dealer (últimos 3 meses cerrados).
  • Priorización → vista de solo lectura: ordena dónde poner el esfuerzo comercial (Pareto por potencial + matriz de segmentos) con estos mismos datos y variables. No modifica nada.
