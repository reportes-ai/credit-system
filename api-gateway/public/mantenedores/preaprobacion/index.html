Políticas de Preaprobación

Variables de la preaprobación automática de créditos — un solo set de reglas para el Portal del Dealer y para el bot de WhatsApp (Facilito).

Condiciones de la operación
Renta y carga financiera
Información comercial (umbrales de bloqueo)
WhatsApp — Facilito
Qué afecta este mantenedor
  • Condiciones de la operación / Renta / Info comercial → preaprobación automática del Portal del Dealer (botón "Pre-aprobar cliente": veredicto PREAPROBADO/REVISIÓN y cuotas ofrecidas).
  • Plazos → plazos que se cotizan en la preaprobación del portal (el bot de WhatsApp usa sus tramos del guion).
  • WhatsApp — severidad máxima y pie exprés → preevaluación del bot Facilito (veredicto de la preevaluación DealerNet y mensaje de trámite exprés).
  • Antigüedad máx. default → solo aplica si la matriz de Política de Crédito no define antigüedad para USADO; si la define, manda la matriz.
