Ranking de Colocaciones

Popup mensual con el podio 1°/2°/3° por créditos otorgados del mes anterior — con fanfarria épica 🎺.

Configuración
Popup activo
Si cae sábado/domingo, mostrar el día hábil siguiente
Música 🎺
El podio muestra el mes ANTERIOR completo.
Se sigue mostrando N días a quienes no se habían conectado (1 vez por mes por persona/navegador).
Usa {mes} para el nombre del mes rankeado (ej. JUNIO 2026).
Qué afecta este mantenedor
  • Popup del podio → aparece en todas las páginas de la app a todos los usuarios conectados desde el día configurado (1 vez al mes por navegador).
  • Ranking → cuenta créditos con estado OTORGADO del mes anterior por ejecutivo (tabla creditos, mismos datos del Dashboard y Comisiones).
  • Día / hábil siguiente / vigencia → controlan cuándo y por cuántos días se muestra.
