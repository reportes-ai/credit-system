Recálculo Programado

Recalcula automáticamente todos los motores (comisiones, ingresos, comisión dealer/parque, seguros) de los meses abiertos, por día y horario.

—
Prende para que el recálculo corra solo según los tramos de abajo.
Tramos horarios
Se evalúan de arriba hacia abajo: el primer tramo que coincida (día + hora actual) manda su intervalo. Si ninguno coincide, se usa el intervalo por defecto.

Qué afecta este mantenedor
  • Recálculo de motores → recalcula comisiones, ingresos, comisión dealer/parque y seguros de los meses abiertos (los cerrados no se tocan). Alimenta el Dashboard, Revisión de Comisiones y Reportería.
  • On/off y tramos → definen cuándo corre solo. Es una red de seguridad: además, cada carga/edición/digitación ya recalcula su mes automáticamente.
  • Recalcular ahora → fuerza una corrida inmediata de todos los meses abiertos.
