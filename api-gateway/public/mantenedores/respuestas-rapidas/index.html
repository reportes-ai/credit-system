Qué afecta este mantenedor

  • Título / Texto / Orden → alimentan el panel Respuestas rápidas a la derecha del chat en Atención Remota. Al pinchar una, el texto se carga en el mensaje (no se envía hasta que el ejecutivo presiona Enviar).
