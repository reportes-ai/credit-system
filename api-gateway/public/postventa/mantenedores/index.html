Correo de Orden de Pago a Contabilidad

Texto del correo que se envía a Contabilidad al pulsar Enviar Correo en Emisión de Orden de Pago (Saldo Precio y Comisión). El correo sale desde la aplicación; el usuario que lo envía queda en copia (CC).

Saldo Precio
Variables: {nOrden} {dealer} {num_op} {financiera} {fecha_otorgado} {fecha_recepcion}
Comisión
Variables: {nOrden} {dealer} {num_op} {financiera} {numero_factura} {fecha_factura} {fecha_recepcion} {doc}
Qué afecta este recuadro
• Destinatario → a quién llega el correo de la orden de pago (Emisión Orden de Pago Saldo Precio y Emisión Orden de Pago Comisión).
• Asunto / Cuerpo / Firma → el texto del correo que arma cada página al pulsar «Enviar Correo». El cuadro de la orden se inserta automáticamente entre el cuerpo y la firma.
