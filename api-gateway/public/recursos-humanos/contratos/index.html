Qué afecta este mantenedor
  • Cargos y descripciones → cartas oferta y contratos de esta página; un cargo sin descripción genera un recordatorio diario (campana) hasta que se cargue.
  • Catálogo de beneficios → los checkbox disponibles al armar cada carta oferta, y las cláusulas que hereda el contrato.
  • Carta ACEPTADA → Contratar → crea el contrato con lo ofertado (fuente única) y deja el documento imprimible para firma.
