Score de Mora — histórico de cartera propia

Tasa de mora real por segmento (ubicación, monto, plazo, pie, cartera) según el historial de pagos de las carteras migradas. Índice 1,0 = mora promedio; 2,0 = el doble. Referencia para volver a colocar cartera AutoFácil y para valorizar ventas de cartera.
