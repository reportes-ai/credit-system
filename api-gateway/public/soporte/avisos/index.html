Qué afecta este mantenedor
  • Mensaje / Sonido / Retardo / Duración → aviso emergente con sonido en todas las páginas de la Suite para todos los usuarios conectados (receptor global /js/aviso-linea.js, sondeo cada 20 s).
  • Permiso de envío → funcionalidad avisos_linea, asignable por perfil en Usuarios y Permisos.
