Qué afecta este mantenedor
  • API key / modo Fintoc → habilita la lectura de saldos y cartolas. Sandbox usa datos ficticios; producción (sk_live) trae las cuentas reales.
  • Cuentas conectadas → cada enlace trae saldo disponible/contable y alimenta la tabla banco_movimientos.
  • Sincronización → refresca saldos y hace UPSERT de movimientos (idempotente). Base para la futura conciliación de pagos en Aplicación de Fondos y Cobranza.
