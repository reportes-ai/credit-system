Conciliación Bancaria

Matchea la cartola del banco contra pagos de cuotas y órdenes de pago del sistema

Mientras Fintoc no esté contratado, la cartola se carga manualmente desde el Excel del banco. El sistema detecta las columnas (Fecha, Cargo/Abono, Descripción) y no duplica al re-cargar el mismo archivo. Cuando Fintoc se active, los movimientos llegarán solos y esta pantalla no cambia.

Qué afecta este módulo
  • Conciliar / desconciliar → solo marca el movimiento bancario; nunca modifica los pagos de cuotas ni las órdenes de pago.
  • Cuentas manuales → viven en la misma tabla que las Conexiones Bancarias (Fintoc); al contratar Fintoc los movimientos llegan solos.
  • Cargas de cartola → quedan registradas en Auditoría (acciones IMPORTAR / CONCILIAR / DESCONCILIAR).
